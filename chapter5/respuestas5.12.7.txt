1. Utilizando funciones de entrada y salida del Lenguaje C: Realiza un programa que solicite un número 
y te imprima todos los números menores a dicho número empezando por el cero. 
Si el número introducido es menor que cero que imprima todos los números que faltan para llegar hasta el cero
    #include <stdio.h>

    int main() {

        int x, i;

        printf("Introduce un numero: ");
        scanf("%d", &x);

        if (x >= 0) {
            for(i = 0; i < x; i++) {
                printf("%d\n", i);
            }
        } 
        else {
            for(i = x; i <= 0; i++) {
                printf("%d\n", i);
            }
        }

        return 0;
    }

2. ¿Es posible imprimir en pantalla con la función scanf?
    No, no es posible imprimir o mostrar datos en pantalla con la función scanf. 
    scanf se utiliza exclusivamente para leer o capturar datos ingresados por el usuario
